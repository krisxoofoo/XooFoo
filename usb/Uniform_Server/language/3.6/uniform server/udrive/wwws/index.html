Uniform Server est un package WAMP permettant t'utiliser un serveur sur un ordinateur basé sur les OS Windows : XP ou Vista. Il est léger à télécharger et mobile à utiliser ; et peux l'être en local ou en ligne. De très nombreux Développeurs l'utilise pour tester leurs applications réalisées avec PHP, MySQL, Perl, ou un serveur HTTPd Apache.
