 Installation
 ------------
 Installer ce package de traduction sur une installation propre d'Uniform Server 5.6.1-Nano ou au dessus

 Translation pack has been packaged in two file formats the “exe” is a 7z
 self-extracting archive while the “zip” archive requires manual extraction.
 Depending on the file format choose one of the following installation methods:

 exe:
  Copier V56_French_1_0.exe dans le dossier UniServer. En double cliquant sur le fichier, vous lancerez l'extraction.
  Autoriser l'écrasement des fichiers existants à l'invite. Pour préserver l'espace du disque dur, supprimer V56_French_1_0.exe.

 zip:
  Créé un nouveau dossier et copier V56_French_1_0.zip dedans, pour extraire tous les fichiers dans ce dernier.
  Naviguer dans le dossier nommé V56_French_1_0 et copier la totalité du contenu dans le répertoire contenant UniServer.
  Autoriser l'écrasement des fichiers existants à l'invite. Pour préserver l'espace du disque dur, supprimer V56_French_1_0.exe.

 -----
 Notes
 -----

 Ce package permet de changer la langue dans PhpMyAdmin et Apanel après configuration dans ces derniers.
 
 UniTray ne prévoient pas de changement de langue, pour basculer de langue que vous devez remplacer le
 fichier UniServer\unicon\tray_menu\UniTray1.ini avec le fichier de langue approprié.
