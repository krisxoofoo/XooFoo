Installation : To install this theme the way it was intended to be installed please upload all contents of the "root" folder to your XOOPS_Root.

Finally go to your system preferences and set this theme as your default theme.

/* french */
Installation : une fois l'archive décompressée, il faut téléverser  le contenu (et uniquement celui-ci) du dossier à la racine de votre site.
Ensuite, dans les préférences de l'administration de votre site, vous sélectionnez ce thème par défaut.
En cas d'utilisation de la surcharge du thème, vous devez cocher dans les préférences de l'administration : l'actualisation du thème et templates à partir du répertoire. Une fois le thème utilisé et les fichiers temporaires créés dans le dossier "templates_c" (génération lors de la première visite sur le template "surchargé", vous pourrez décocher cette case.
